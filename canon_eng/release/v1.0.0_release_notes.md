EDCMBone v1.0.0 — Release Notes

Release: 1.0.0
Status: Machine-Complete
Authority: edcmbone_bundle_manifest_v1.json
Date: Initial canonical freeze


---

Summary

EDCMBone v1.0.0 marks the first fully deterministic, versioned release of:

Energy–Dissonance Circuit Model (EDCM)

Prime Circular Neural Architecture (PCNA)

Prime Circular Tensor Architecture (PCTA)


This release transitions EDCMBone from conceptual framework to implementation-ready specification.

All structural rules, inventories, normalization policies, and output schemas required for reproducible implementation are now frozen.


---

Major Components in v1.0.0

1. Parsing Canon (Frozen)

Actor → Utterance → Turn → Round pipeline

Initiator-return round closure rule

SYS/TOOL exclusion from rounds (default)

UNK actor handling

Explicit turn boundary rules

Lossless reconstruction guarantee


Deterministic procedural derivation is included.


---

2. Operator-EDCM (Frozen)

Schema: Option A Families

Family	Meaning

P	Position / relational anchors
K	Connectors
Q	Qualifiers / scope
T	Temporal / modality
S	Self / referential


Frozen Rules

Whole-word precedence over affix stripping

Longest-match-first affix resolution

Collision priority: P > K > Q > T > S

Hyphen compound emits exactly 1 K

Question mark emits exactly 1 Q

Em dash emits 0 operator tokens

Slash ignored in v1

Aggregation method: sum-and-renormalize

Operator window: k = 8 turns

Rolling stride = 1


Closed-class inventory and affix inventory are versioned.


---

3. Behavioral-EDCM (Frozen v1)

9 Metrics

C, R, D, N, L, O, F, E, I

Deterministic Features

Structural-first marker sets

Tier A (structural) preferred

Tier B lexical anchors explicitly quarantined

Saturating caps defined per metric

Fixation computed via structural cosine proxy

Escalation computed via positive derivative

Integration Failure computed via correction–acknowledgement proxy

Behavioral window: m = 4 closed rounds

Rolling stride = 1


All formulas and caps frozen.


---

4. Bridge Layer (Frozen)

Pearson correlation method

L1 magnitude divergence detection

Default divergence threshold = 0.20

Adjustable threshold permitted (configurable)

Open rounds excluded

Bridge is strictly read-only

5. PCNA Topology (Frozen)

Prime count N = 53

Circular embedding

Edge layers:

Ring adjacency

Heptagram steps (2, 3, 5)

Twin-prime edges

Band edges (7, 11, 13)


Combined weights summed across layers

Row-normalized routing kernel
