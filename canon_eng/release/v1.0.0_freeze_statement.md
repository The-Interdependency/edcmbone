Declaration

EDCMBone v1.0.0 is hereby declared structurally frozen and implementation-deterministic.

This release defines a complete, versioned, machine-executable specification for:

Lossless transcript parsing

Turn-native Operator-EDCM (bones-only)

Round-native Behavioral-EDCM (9 metrics)

Observational Bridge alignment

Prime-indexed PCNA topology (N = 53)

PCTA transport interface (consumptive, non-modifying)


All invariants, inventories, thresholds, and normalization rules required for deterministic implementation are explicitly declared.

No additional theoretical refinement is required for coding to begin.


---

What “Frozen” Means

For v1.0.0:

Operator family schema (Option A) is fixed.

Closed-class inventory is versioned.

Affix inventory and stripping policy are versioned.

Window sizes and stride are fixed.

Divergence thresholds default to 20%.

PCNA prime count (N = 53) is fixed.

Edge-layer definitions and normalization rules are fixed.

All output schemas are versioned and binding.

Behavioral caps and normalization functions are frozen.


Changes to any of the above require a version increment as defined in versioning_policy.md and edcmbone_bundle_manifest_v1.json.

Determinism Guarantee

Given:

Identical input transcript

Identical parsing implementation honoring tokenization invariants

Identical inventories and policies

Identical window parameters


Two independent implementations SHALL produce identical:

Turn sets

Round sets

Operator vectors

Behavioral vectors

Bridge outputs


Floating-point tolerance: ±1e-6.


---

Architectural Guarantees

1. Losslessness

Raw transcript text is never modified or discarded.

2. Layer Independence

Operator does not modify Behavioral.

Behavioral does not modify Operator.

Bridge is read-only.

PCNA/PCTA consumes state but does not mutate upstream metrics.


3. Structure First

Operator-EDCM discards open-class lexical roots and operates strictly on structural bones.

4. Explicit Drift Containment

All known semantic approximations are quarantined in Tier B marker sets and clearly labeled.
